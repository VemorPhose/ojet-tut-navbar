Dataset and visualization method

The laptop-data.csv fields are parsed into robust numeric dimensions for JET charts, tables, filters, and the three animated 3D scenes.

Parsed fields

RAM and weight are read from GB and kg strings. CPU speed is read from GHz values. Storage is summed across GB and TB memory tokens. Pixel density is calculated from the final resolution token in ScreenResolution and the screen size in inches.

 Toolkit split

Oracle JET powers routing, navigation, tab selection, switcher panels, filters, charts, list views, progress, and the table. Three.js renders the draggable 3D point clouds inside those JET-managed panels, using log price axes and 4th-96th percentile bounds to prevent outliers from compressing the scene.
